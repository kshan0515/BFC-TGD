# 🔴 부천 축구동 (BFC-TGD)

부천FC 1995 서포터, 팬들을 위한 인스타그램 및 유튜브 통합 소셜 미디어 허브 서비스입니다. ⚽

## ⚽ 프로젝트 목적 (Purpose)

**부천 축구동**은 서포터와 팬들이 흩어져 있는 구단 소식을 한곳에서 쉽고 빠르게 확인하며 소통할 수 있도록 돕기 위해 개발되었습니다. 

본 프로젝트는 오픈 소스 생태계를 지지하며, 이 코드가 다른 개발자들에게 좋은 학습 자료나 영감이 되기를 바랍니다. 부천FC 팬이라면 누구나 자유롭게 사용하고 기여하실 수 있습니다.

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: Next.js 15+ (App Router), React 19
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **Database**: MongoDB Atlas
- **Scraping**: Python
- **Automation**: GitHub Actions

## ⚙️ 시작하기 (Getting Started)

### 환경 변수 설정
`.env.local` 파일을 생성하고 다음 정보를 입력하세요:
```env
NEXT_PUBLIC_API_URL=your_backend_url
MONGO_URI=your_mongodb_atlas_uri
APIFY_TOKEN=your_apify_token
YOUTUBE_API_KEY=your_youtube_key
```

### 설치 및 실행
```bash
npm install
npm run dev
```
